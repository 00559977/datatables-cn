selectNone

起始版本号: {{ page.since }}

取消所选择的项

注意：这个需要使用 Select扩展的支持

说明(Description)

取消所有选择的项目，这个不受当前选择模式的控制，只要是触发了，所有选择的都会被取消选择。

示例

Datatables初始化：显示全选和取消全选按钮，按钮的文字可以由 language对象配置： 

$('#myTable').DataTable( {
    buttons: [
        'selectAll',
        'selectNone'
    ],
    language: {
        buttons: {
            selectAll: "全选",
            selectNone: "全不选"
        }
    }
} );
