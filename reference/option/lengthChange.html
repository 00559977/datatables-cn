是否允许用户改变表格每页显示的记录数(lengthChange)

起始版本号: 1.10

是否允许最终用户改变表格每页显示的记录数

说明

如果pagination启用了, 这个选项设为true会显示一个可以让用户选择每页显示多少条记录的下拉框。 下拉框中的选项值利用lengthMenuDT 来配置。

注意：默认情况下这个下拉框在表格的左上角显示。在哪里显示可以通过domDT 和 CSS控制。

另外，如果pagination使用pagingDT 配置成禁用，这个选项就会自动被设为false。因为如果pagination被禁用，这个选项就没有存在的合理性了。

数据类型

This option can be given in the following type(s):

  • booleanJS

默认值

  • Value: true

示例

Disable user ability to change number of records per page:

$('#example').dataTable( {
  "lengthChange": false
} );
