info

Since: 1.10

控制是否显示表格的信息

说明

当应用了搜索时，会在适当位置显示表格的搜索条件。 这个配置允许打开或者关闭这个特性。

需要注意的是，默认情况下信息提示显示在表格下面的左边，但是这个可以用domDT 和 CSS来控制.

数据类型

这个选项用下列的数据类型赋值:

  • booleanJS

默认值

  • Value: true

示例

Disable information display:

$('#example').dataTable( {
  "info": false
} );
