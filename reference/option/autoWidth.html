自动宽度(autoWidth)

Since: 1.10

控制DataTables列宽的智能控制

说明

启用或者禁止自动列宽的计算。如果已经定义了列宽，禁止这个选项可以实现最优的性能（因为这个选项会耗费一点的时间去计算列宽）。 columns.widthDT.

数据类型

这个选项用下列的数据类型赋值。

  • booleanJS

默认值

  • Value: true

示例

禁止智能计算列宽:

$('#example').dataTable( {
  "autoWidth": false
} );
