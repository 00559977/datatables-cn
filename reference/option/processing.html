处理状态(processing)

起始版本号: 1.10

是否显示正在处理的状态。

说明

当表格处在处理过程（例如排序）中时，启用或者禁止 'processing'指示器的显示。当处理大数据时，处理过程耗费的时间很明显，这个功能就显得非常有用。

数据类型

This option can be given in the following type(s):

  • booleanJS

默认值

false

示例

Disable pagination:

$('#example').dataTable( {
	"processing": true
} );
