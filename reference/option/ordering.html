排序(ordering)

起始版本号: 1.10

是否允许DataTables排序。

说明

允许或禁止对各个数据列使用排序。 默认情况下，点击每列的表头单元格,可以根据该列对整个表格进行排序。这个选项可以停用这个排序功能。

注意：禁止或者启用单个列的排序功能，可以通过该列的columns.orderableDT选项来实现。 现在的这个ordering选项是一个全局的配置- 如果禁止，整个DataTables都会停用排序功能。

数据类型

This option can be given in the following type(s):

  • booleanJS

默认值

true

示例

Disable ordering in the table:

$('#example').dataTable( {
  "ordering": false
} );
