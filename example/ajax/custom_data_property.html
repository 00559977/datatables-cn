当从 Ajax 源加载数据，在默认情况下，数据表将寻找要使用返回的对象的数据参数中的数据 （例如 {"data"： [......]}）。 通过使用 ajaxDT选项的dataSrc 启动选项来改变属性。

dataSrc有许多用法如下：

  • 作为一个字符串，如 dataSrc: 'myData'，下面的例子展示的是这个用法
  • 为空字符串，如 dataSrc:""
  • 作为一个方法，如 dataSrc： function(json) {} （您可以从 XML 转换到一个 Javascript 对象）

$(document).ready(function() {
        $('#example').dataTable( {
            "ajax": {
                "url": "data/arrays_custom_prop.txt",
                //默认为data，这里定义为demo
                "dataSrc": "demo"
            }
        } );
    } );
