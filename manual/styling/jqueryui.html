jQuery UI

jQueryUI 是一套 jQuery 的页面 UI 插件，包含很多种常用的页面空间，例如 Tabs（如本站首页右上角部分） 、拉帘效果（本站首页左上角）、对话框、拖放效果、日期选择、颜色选择、数据排序、窗体大小调整等等非常多的内容。

同Bootstrap和Foundation一样，使用jQuery UI也只需引入css和js即可

css: http://cdn.datatables.net/plug-ins/28e7751dbec/integration/jqueryui/dataTables.jqueryui.css

js: http://cdn.datatables.net/plug-ins/28e7751dbec/integration/jqueryui/dataTables.jqueryui.js

你也可以在github上clone下来 https://github.com/DataTables/Plugins/tree/master/integration/jqueryui
